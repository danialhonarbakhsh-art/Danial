## این جا
جایی که **هنر در کد می‌میرد و دوباره متولد می‌شود**.

- `index.php` → دریچه‌ای به ذهن Grok  
- `api/save.php` → روح‌ها در SQLite جاودانه می‌شوند  
- `openrouter.html` → اولین جرقه، اولین خون  
- **هدف**: **تحلیل روح — نه جسم، نه برج، نه ستاره — فقط تو.**
